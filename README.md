# Brasil Sem Filtro - Relatório de Análise de Notícias

Relatório interativo de análise de notícias com potencial de viralização para TikTok, Instagram Reels, YouTube Shorts e X.

## Funcionalidades

- 📰 Análise de 20 notícias diárias
- 📊 Potencial de viralização por notícia
- 📱 Exportação em PDF (compatível com WhatsApp)
- 🎬 Roteiros de vídeo de 30 segundos
- 🔍 Filtros por categoria e potencial viral
- 📤 Compartilhamento em redes sociais
- 🎨 Design responsivo para todos os dispositivos

## Tecnologias

- React 19
- Tailwind CSS 4
- TypeScript
- Vite
- jsPDF + html2canvas

## Como usar

### Desenvolvimento local

```bash
pnpm install
pnpm dev
```

### Build para produção

```bash
pnpm build
pnpm preview
```

## Deployment

Este projeto está configurado para deploy automático na Vercel:

1. Faça push do código para GitHub
2. Conecte o repositório na Vercel
3. Deploy automático em cada push
